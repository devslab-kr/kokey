kokey

한↔영 자판 변환 (두벌식 ↔ QWERTY) · Korean ↔ English keyboard layout converter

TypeScript-first · zero-dependency · ESM / CJS / <script>

Live demo — 직접 쳐보세요

enToKo — 영타를 한글로

영문 자판으로 그대로 치면 한글이 조합됩니다. Try dkssudgktpdy

koToEn — 한타를 영문으로

한글 IME가 켜져 있어도 QWERTY로 복원됩니다. Try ㅇㄴㅁ쇼2068601

Converter — 양방향 변환기

아래에 아무 텍스트나 붙여넣으면 두 방향 결과를 동시에 보여줍니다.

Install

# npm
npm install @devslab/kokey

# CDN — no build step
<script src="https://cdn.jsdelivr.net/npm/@devslab/kokey/dist/kokey.global.js"></script>

Usage

import { koToEn, enToKo, observe } from '@devslab/kokey'

koToEn('안녕')   // 'dkssud'
enToKo('dkssud') // '안녕' — 겹모음·겹받침·받침 넘김까지 실제 IME 그대로

// DOM: data-hangul 인풋을 전부 자동 바인딩
observe() // <input data-hangul="ko|en">
